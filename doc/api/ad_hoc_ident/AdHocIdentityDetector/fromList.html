Creates a new AdHocIdentityDetector, which passes the input to all supplied detectors.

If executeParallel is set to false, the detectors will receive the input one after another, returning after the first AdHocIdentityDetector detects an AdHocIdentity. If executeParallel is set to false, all detectors are run in parallel, returning once all have finished processing. When executing in parallel, the results are evaluated in the same order of the detectors, returning the first detected AdHocIdentity or an exception if any detector threw one.
